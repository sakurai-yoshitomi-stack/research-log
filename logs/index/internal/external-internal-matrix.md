# 外向け × 内側 紐付けマトリックス（β版）
※ 内部運用用 / 更新・書き換え可

本ドキュメントは、日本構造科学研究所における  
**外部向け研究テーマ（external）** と  
**内部研究コア・進化レイヤー（internal）** の  
対応関係を可視化するためのマトリックスである。

本マトリックスは「説明用」ではなく、  
以下を目的とする。

- 外向け記述が中核を侵食していないかの確認
- 新規テーマ出現時の所属レイヤー判定
- 研究の宗教化・思想化・過剰一般化の防止

## 内部メモ（超重要）

- 外向けテーマは **研究対象の列挙** であり、研究の核ではない  
- 研究の核は常に  
  **構造 / 順序 / 間 / 反転** にある  
- 本マトリックスは完成させない  
- 研究の進化に応じて「書き換わること」自体が価値
